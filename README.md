# 📺 Ratings de Séries com OMDb API

Este projeto é uma aplicação web que consome a API OMDb para exibir os ratings dos episódios de uma série em formato de matriz.

## 🚀 Funcionalidades

- Exibe nome e poster da série
- Mostra episódios organizados em matriz:
  - Linhas = temporadas
  - Colunas = episódios
- Cada célula contém o rating do episódio

## 🛠 Tecnologias

- HTML
- CSS
- JavaScript (Fetch API)
